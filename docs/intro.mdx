# Introduction

`@open-kerno/commons` is a shared TypeScript utility library for Node.js applications. It provides a set of well-tested, typed modules that solve common problems across services.

## Installation

```bash
npm install @open-kerno/commons
```

## Modules

| Module | Description |
|---|---|
| [`http`](./http) | `HttpStatusCode` and `HttpMethod` enums |
| [`errors/http`](./errors#http-errors) | Typed HTTP error classes |
| [`errors/database`](./errors#database-errors) | Typed database error classes |
| [`collections`](./collections) | Array to Map/Set utilities |
| [`math/distribution`](./math#distribution) | Weighted value distribution |
| [`math/rounding`](./math#rounding) | Precision rounding |
| [`logger`](./logger) | Structured logger with field masking |
| [`infrastructure/database/postgres`](./infrastructure) | PostgreSQL connection pool wrapper |
| [`object`](./object) | JSON encoding helpers |
| [`currency`](./currency) | Immutable, fluent money arithmetic |

## Requirements

- Node.js 18 or higher
- TypeScript 5 or higher
