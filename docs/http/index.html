HTTP

The http module provides two enums: HttpStatusCode for standard HTTP response codes, and HttpMethod for standard HTTP request methods.

Import​

import { HttpStatusCode, HttpMethod } from '@open-kerno/commons/http';

Example​

import { HttpStatusCode } from '@open-kerno/commons/http';

res.status(HttpStatusCode.CREATED).json({ id: newUser.id });
res.status(HttpStatusCode.NOT_FOUND).json({ message: 'User not found' });

HttpMethod​

An enum with all standard HTTP request methods.

MemberValue
GET'GET'
POST'POST'
PUT'PUT'
PATCH'PATCH'
DELETE'DELETE'
HEAD'HEAD'
OPTIONS'OPTIONS'
TRACE'TRACE'
CONNECT'CONNECT'

Example​

import { HttpMethod } from '@open-kerno/commons/http';

const allowedMethods = [HttpMethod.GET, HttpMethod.POST];

if (!allowedMethods.includes(req.method as HttpMethod)) {
  res.status(405).end();
}
